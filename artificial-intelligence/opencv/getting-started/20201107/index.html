环境搭建可参考：B 站视频

# 知识点

  • 读取图像：imread
  • 显示图像：imshow

# C++ 代码

#ifndef DAY01
#define DAY01

#include <opencv2/opencv.hpp>
#include <iostream>

using namespace std;
using namespace cv;

void day01() {

	Mat src = imread("E:\\_Image\\OpenCVTest\\girl.jpg");

	if (src.empty()) {
		cout << "could not load image.." << endl;
		return;
	}

	namedWindow("show", WINDOW_AUTOSIZE);
	imshow("show", src);

	waitKey(0);
}

#endif // !DAY01

注意事项：
在 VS2017 中创建项目的时候选择创建空项目，预编译标头取消勾选。选择 x64 进行编译。
