# 前言

适用于从一个 git 项目中，将体积较大的资源彻底从 git 中删除，包括历史提交记录。
如果仅仅在目录中删除一个文件是不够的，只要在提交记录中有这个文件，那么 .git 中就会有这个文件的信息。
用 filter-branch 可以强制修改提交信息，将某个文件的历史提交痕迹也抹去，就像从来没有过这个文件一样。
