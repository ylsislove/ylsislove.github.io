# 环境

  • JRE（Java Runtime Environment）
  • STM32CubeMX v6.3.0
  • HAL 库 L4 v1.7.0
  • Keil5-MDK

环境搭建参考：STM32CubeMX 学习记录 -- 安装配置与使用
