йҰ–йЎө ж•°жҚ®еә“ PostgreSQL

# еүҚиЁҖ

ж¶үеҸҠеҲ° PostgreSQL 10.0 ж•°жҚ®еә“е’Ң postgis 2.4 ж•°жҚ®еә“жү©еұ•пјҢеҹәдәҺ docker е®үиЈ…пјҢеңЁжӯӨиҝӣиЎҢжҖ»з»“е’Ңи®°еҪ•пҪһ

# еҲӣе»әе‘Ҫд»Ө

docker run --name postgis -d -e POSTGRES_USER=postgres -e POSTGRES_PASS=postgres -e POSTGRES_DBNAME=gis -e ALLOW_IP_RANGE=0.0.0.0/0 -p 5432:5432 -v /home/wy/postgre_data:/var/lib/postgresql -v /home/wy/server/cityscene/data:/data --restart=always kartoza/postgis:10.0-2.4

еҸӮж•°иҜҙжҳҺпјҡ

  • --name postgisпјҡе‘ҠиҜү docker ж–°е»әдёҖдёӘй•ңеғҸпјҢе‘ҪеҗҚдёә postgis
  • -dпјҡи®©е®№еҷЁеҗҺеҸ°иҝҗиЎҢ
  • -eпјҡи®ҫзҪ®е®№еҷЁеҶ…зҺҜеўғеҸҳйҮҸгҖӮ POSTGRES_USER и®ҫзҪ®и¶…зә§з”ЁжҲ·пјӣ POSTGRES_PASS и®ҫзҪ®з”ЁжҲ·еҜҶз Ғпјӣ POSTGRES_DBNAME и®ҫзҪ®еҲқе§ӢеҢ–з©әй—ҙж•°жҚ®еә“зҡ„еҗҚеӯ—пјӣ ALLOW_IP_RANGE е…Ғи®ёд»»дҪ•е®ўжҲ·з«ҜиҝһжҺҘеҪ“еүҚз”ЁжҲ·
  • -pпјҡи®ҫзҪ®з«ҜеҸЈгҖӮеҶ’еҸ·е·Ұиҫ№жҳҜжң¬жңәеҗ‘еӨ–жҡҙйңІзҡ„з«ҜеҸЈпјҢеҶ’еҸ·еҸіиҫ№жҳҜе®№еҷЁеҶ…йғЁж•°жҚ®еә“жңҚеҠЎз«ҜеҸЈ
  • -vпјҡеҲӣе»әж•°жҚ®еҚ·пјҢдҪҝж•°жҚ®жҢҒд№…еҢ–еӯҳеӮЁгҖӮеҶ’еҸ·е·Ұиҫ№жҳҜжң¬жңәи·Ҝеҫ„пјҢеҶ’еҸ·еҸіиҫ№жҳҜжғіиҰҒеӯҳеӮЁзҡ„е®№еҷЁеҶ…йғЁж•°жҚ®
  • --restart=always еҲӣе»әдёҖжқЎиҮӘеҠЁеҗҜеҠЁзҡ„и§„еҲҷпјҢе‘ҠиҜү Docker пјҢ жҜҸж¬ЎеҗҜеҠЁзҡ„ж—¶еҖҷ postgis е®№еҷЁиҮӘеҠЁеҗҜеҠЁ

Noteпјҡе®һйҷ…дёҠпјҢз¬¬дәҢдёӘ -v дёҚжҳҜеҝ…иҰҒзҡ„пјҢжҲ‘иҝҷйҮҢж·»еҠ еҸӘжҳҜеӣ дёәжҲ‘ејҖеҸ‘зҡ„еә”з”ЁзЁӢеәҸжңүиҝҷдёӘйңҖжұӮ

е‘Ҫд»ӨиҝҗиЎҢе®ҢеҗҺпјҢз”Ё docker ps е’Ң docker logs postgis жҹҘзңӢе®№еҷЁжҳҜеҗҰеҲӣе»әжҲҗеҠҹпјҢд»ҘеҸҠеҲӣе»әиҝҮзЁӢдёӯзҡ„ж—Ҙеҝ—

# з”Ё pgAdmin иҝһжҺҘж•°жҚ®еә“

дёӢиҪҪй“ҫжҺҘпјҡpgAdmin

е®үиЈ…е®ҢжҲҗеҗҺжү“ејҖпјҢйҰ–е…Ҳдјҡи®©дҪ и®ҫзҪ®дёҖдёӘ pgAdmin зҡ„и®ҝй—®еҜҶз ҒпјҢеҸҜд»Ҙи®ҫзҪ®дёә postgres еҘҪи®°

з„¶еҗҺзӮ№еҮ»ж·»еҠ ж–°жңҚеҠЎпјҢиҫ“е…ҘеҲ«еҗҚпјҢеңЁ Connection дёӯиҫ“е…Ҙ ip ең°еқҖе’Ңе…¶д»–дҝЎжҒҜпјҢзӮ№еҮ»дҝқеӯҳпјҢдёҚжҠҘй”ҷиЎЁзӨәиҝһжҺҘжҲҗеҠҹ

# йҖҡиҝҮ PostGIS Shapefile Import/Export Manager еҜје…ҘзҹўйҮҸж•°жҚ®

иҝҷдёӘе·Ҙе…·жҳҜзҹўйҮҸ Shapefile ж–Үд»¶зҡ„еҜје…Ҙе·Ҙе…·гҖӮдёӢиҪҪй“ҫжҺҘпјҡPostGISпјҢеҸҜд»ҘдёӢиҪҪжңҖж–°зүҲжң¬зҡ„ 3.1.4

е®үиЈ… PostGIS Shapefile Import/Export Manager иҝҷдёӘе·Ҙе…·д№ӢеүҚйңҖиҰҒеңЁжң¬ең°е®үиЈ… PostgreSQL 10пјҢе®үиЈ…иҝҮзЁӢеҸӮиҖғпјҡPostgreSQL 10 е®үиЈ…иҜҰз»ҶжӯҘйӘӨ

еӣ дёәжҲ‘д»¬е№¶дёҚйңҖиҰҒжң¬ең°зҡ„ж•°жҚ®еә“пјҢжүҖд»Ҙе®үиЈ…ж—¶еҸҜд»ҘеҸӘеӢҫйҖү server е’Ң command line иҝҷдёӨйЎ№еҚіеҸҜгҖӮ

е®үиЈ…е®ҢжҲҗеҗҺпјҢе°ұеҸҜд»Ҙе®үиЈ…еҲҡеҲҡдёӢиҪҪзҡ„ Shapefile ж–Үд»¶еҜје…Ҙе·Ҙе…·дәҶпјҢе®үиЈ…иҝҮзЁӢеҸҜд»ҘеҸӮиҖғ PostGIS 2.5.0 е®үиЈ…иҜҰз»ҶжӯҘйӘӨ

еҗҢзҗҶпјҢеӣ дёәжҲ‘д»¬дёҚдҪҝз”Ёжң¬ең°зҡ„ PostgreSQL ж•°жҚ®еә“пјҢжүҖд»Ҙе®үиЈ…иҝҮзЁӢдёӯеҸҜд»ҘйҖүжӢ©дёҚе®үиЈ… spatial database з©әй—ҙж•°жҚ®еә“гҖӮжңҖеҗҺе®үиЈ…е®ҢжҲҗ

жү“ејҖ PostGIS Shapefile Import/Export ManagerпјҢзӮ№еҮ» view connection detail пјҢиҫ“е…ҘиҝһжҺҘдҝЎжҒҜпјҢеҰӮдёӢеӣҫ

зӮ№еҮ» OK еҗҺпјҢдёӢйқўиҫ“еҮә successпјҢиЎЁзӨәиҝһжҺҘжҲҗеҠҹ

жҺҘдёӢжқҘе°ұжҳҜеҜје…ҘзҹўйҮҸж•°жҚ®дәҶпјҢжҲ‘д»¬д»ҺзҪ‘дёҠдёӢиҪҪдёҖд»Ҫе…ЁеӣҪ 62 дёӘеҹҺеёӮе»әзӯ‘зү©зҡ„зҹўйҮҸж•°жҚ®пјҢдёӢиҪҪй“ҫжҺҘпјҡе…ЁеӣҪ 62 дёӘеҹҺеёӮе»әзӯ‘зү©иҪ®е»“зҹўйҮҸж•°жҚ®пјҢжҲ‘д»¬и§ЈеҺӢдёӘжӯҰжұүзҡ„е»әзӯ‘зү©ж•°жҚ®пјҢз„¶еҗҺжҠҠи§ЈеҺӢеҮәжқҘзҡ„ж•°жҚ®ж”ҫеҲ°дёҖдёӘжІЎжңүдёӯж–Үеӯ—з¬Ұзҡ„и·Ҝеҫ„дёӢпјҢи·Ҝеҫ„д№ҹдёҚиҰҒиҝҮй•ҝпјҢеҗҢж—¶е…ЁйғЁйҖүдёӯпјҢжҢү F2 е°Ҷдёӯж–ҮеҗҚз§°ж”№жҲҗиӢұж–Үзҡ„пјҢеҰӮдёӢеӣҫ

з„¶еҗҺеңЁеҜје…Ҙе·Ҙе…·дёӯзӮ№еҮ» Add FileпјҢйҖүжӢ©еҲҡеҲҡи§ЈеҺӢеҮәжқҘзҡ„ wuhan.shp пјҢе°ұеҸҜд»ҘзңӢеҲ°еңЁ Import List йҮҢеҮәзҺ°дәҶжҲ‘д»¬йҖүжӢ©зҡ„ж–Үд»¶пјҢеҸҜд»ҘеңЁиҝҷйҮҢжӣҙж”№иЎЁеҗҚзӯүдҝЎжҒҜпјҢзЎ®е®ҡж— иҜҜеҗҺзӮ№еҮ»дёӢж–№зҡ„ Import жҢүй’®пјҢеҰӮдёӢеӣҫ

еҸҜд»ҘзңӢеҲ°дёӢж–№зҡ„иҫ“еҮәжЎҶдёӯеҮәзҺ° import completedпјҢиЎЁзӨәеҜје…ҘжҲҗеҠҹгҖӮ

жҲ‘д»¬иҝҳеҸҜд»ҘеңЁ pgAdmin дёӯжҹҘзңӢеҲ°еҲҡеҲҡеҜје…Ҙзҡ„ж•°жҚ®пјҢеҰӮдёӢеӣҫ

# йҖҡиҝҮ QGIS еҜје…ҘзҹўйҮҸж•°жҚ®

дёҠйқўйӮЈдёӘж–№жі•еҝ…йЎ»иҰҒеңЁжң¬жңәдёҠе®үиЈ… PostgreSQL ж•°жҚ®еә“пјҢжүҚиғҪе®үиЈ… PostGIS еҜје…Ҙе·Ҙе…·пјҢеҚідҪҝжҲ‘д»¬з”ЁдёҚеҲ°жң¬жңәзҡ„ж•°жҚ®еә“пјҢжүҖд»Ҙз¬¬дәҢз§ҚеҜје…Ҙж–№ејҸе°ұжҳҜйҖҡиҝҮ QGIS иҝһжҺҘ Docker е®№еҷЁеҶ…зҡ„ PostgreSQL ж•°жҚ®еә“е№¶еҜје…Ҙж•°жҚ®пјҢиҝҷж ·е°ұдёҚйңҖиҰҒеңЁжң¬жңәдёҠе®үиЈ… PostgreSQL е’Ң PostGIS дәҶгҖӮ
