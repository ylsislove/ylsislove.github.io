# 前言

最近再搞农业物联网的项目，用到了 ZigBee 协议栈，需求之一是要监测土壤温度的数据，便采购了「 善学坊 」家的「 YTWD-A1 」传感器，其本质是 DS18B20 芯片，但网上关于这个芯片的数据读取方式多是 51 单片机或 STM32 相关的代码，经过自己的一番踩坑和实验，终于在 ZigBee 协议栈下写好了这个芯片的驱动，便在此记录下来，希望能帮助到有需要的小伙伴～
